# Arquitectura Full-Stack

- Frontend: Angular standalone components, 30 pantallas con plantillas diferenciadas por tipo de flujo.
- Backend: Spring Boot REST, JPA/JdbcTemplate y validaciones de entrada.
- Database: PostgreSQL con 40 tablas de dominio y relaciones visibles.
- Runtime local: Docker Compose con frontend, backend y postgres.
- Gate anti-falso: no pasan endpoints decorativos, tablas genericas ni UI clonada.
