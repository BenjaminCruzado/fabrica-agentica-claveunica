# React Handoff

Contrato recomendado para apps React generadas:

- `src/app` o `src/pages`: rutas principales.
- `src/components`: componentes visuales reutilizables.
- `src/features`: modulos de dominio.
- `src/lib`: clientes API, validadores y helpers.
- `src/styles`: estilos globales.

La app final se escribe en `app-generada/` y debe incluir su propio `README.md`, scripts de ejecucion y archivos Docker.
