# Frontend Agents Policy

Todo agente que genere UI debe cumplir:

- Usar esta plantilla como base conceptual.
- No crear una landing decorativa si el usuario pidio una aplicacion.
- Mantener componentes separados de reglas de negocio.
- Registrar pantallas generadas en la matriz de trazabilidad.
- Dejar evidencia visual o checklist de UI cuando existan pruebas E2E.
