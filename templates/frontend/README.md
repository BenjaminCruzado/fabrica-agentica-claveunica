# Frontend Template

Plantilla base obligatoria para apps generadas por la fabrica.

Reglas:

- La primera pantalla debe ser usable, no solo portada.
- Debe existir navegacion principal clara.
- Debe incluir estados de carga, vacio, error y exito.
- Formularios con labels visibles y mensajes de validacion.
- Tablas con busqueda, filtros, paginacion o equivalente cuando aplique.
- Accesibilidad minima: foco visible, contraste suficiente y botones con texto comprensible.
