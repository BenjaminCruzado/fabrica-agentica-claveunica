# Constitucion De La Fabrica

La Fabrica Agentica Evaluacion Final es un sistema de proceso para convertir un caso realista en una aplicacion web verificable. Su responsabilidad no es solo generar codigo, sino gobernar el ciclo completo: requisitos, plan, diseno, validacion de rubrica, implementacion, pruebas, evidencia y cierre.

## Estados

- `complete`: fase terminada con evidencia.
- `needs_user_input`: falta decision del estudiante.
- `not_answerable`: falta evidencia suficiente.
- `error`: falla bloqueante.

## Source Of Truth

1. Pauta de evaluacion final.
2. Este documento de constitucion.
3. `FACTORY.md`.
4. `AGENTS.md`.
5. Work Order del caso.
6. Especificacion del profesor.
7. Documentos generados por la fabrica.
8. Codigo generado.
9. Pruebas y evidencia.
