# Reglas De Rubrica Final

La fabrica debe controlar estos minimos:

| elemento | minimo |
|---|---:|
| casos de uso | 10 |
| funcionalidades o flujos | 30 |
| tablas | 40 |
| endpoints API | 40 |
| pantallas | 30 |
| reglas de negocio | 60 |
| validaciones/CHECK | 100 |

Si falta un minimo, el estado correcto es `error` o `needs_user_input`, nunca `complete`.

La validacion principal se registra en `scope-validation.json`.
